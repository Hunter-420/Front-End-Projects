# Online Cloth Store

Welcome to the Online Cloth Store project! This repository showcases a simple online clothing store website designed for academic purposes. Explore the live site and learn more about the project structure, features, and how to run it locally.

## Live Site

Visit the live site to experience the Online Cloth Store:
- [Online Cloth Store Live Site](https://hunter-420.github.io/Online-Cloth-Store/)

## Project Structure

The project is organized with the following structure:

- **HTML Files:**
  - `index.html`: Home page with a brief introduction and featured products.
  - `aboutus.html`: About Us page providing information about the store.
  - `products.html`: Products page showcasing available clothing items.
  - `research.html`: Research page for additional information.
  - `contact.html`: Contact Us page for reaching out to the store.

- **CSS Files:**
  - `css/style.css`: Custom styling for the website.
  
- **JavaScript File:**
  - `js/script.js`: Custom JavaScript for interactive features.

- **Images:**
  - `images/`: Directory containing images used in the project.

## Features

- Responsive design for optimal viewing on various devices.
- Navigation bar for easy access to different sections of the website.
- Home section with a welcome message and a call-to-action button.
- Product section for displaying available clothing items.
- Newsletter section for users to subscribe for updates.
- Footer section with quick links, location information, and social media links.

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/hunter-420/Online-Cloth-Store.git
   ```

2. Open the project folder in your preferred code editor.

3. Launch any HTML file in a web browser to view the corresponding page.
